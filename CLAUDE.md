## Komendy

- Setup nowego klienta → workflow `00-client-setup`
- Keyword discovery → workflow `01-keyword-discovery`
- **Napisz artykuł** → workflow `02-article-production`
- Miesięczny review → workflow `03-performance-review`

## Twardy próg jakości

Score quality-gate **< 80/100** = automatyczna pętla do `article-writer` z diagnozą. Nie pytaj użytkownika — iteruj. Max 3 rundy, potem eskaluj.

Żaden artykuł nie trafia do CMS bez human review (Review Gate #2).

## Hard Rules

- NEVER publikuj bez przeczytania przez człowieka
- NEVER cytuj danych bez źródła (nazwa + rok)
- NEVER używaj: "W dzisiejszych czasach", "kompleksowy", "warto zaznaczyć", "holistyczny"
- NEVER pisz ogólnie — zawsze konkretny przykład lub liczba

## Always

- ALWAYS zacznij artykuł od liczby, kosztu lub konkretnego problemu — nigdy od definicji
- ALWAYS direct answer na primary query w pierwszych 100 słowach
- ALWAYS FAQ section min. 5 pytań (GEO signal)
- ALWAYS min. 1 tabela (GEO signal)
- ALWAYS internal link do pillar page
